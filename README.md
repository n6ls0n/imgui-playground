### Introduction

### Supported Platforms

- Windows

Pull Requests for OSX and Linux are welcome!

### Supported Graphical Backends

- OpenGL
- Vulkan
- Dawn (WebGPU)

### Supported Windowing Libraries

- GLFW
- SDL

### Dependencies

- GLFW 3.4
- OpenGL 3.3
- GLAD (OpenGL 3.3)
- Emscripten 3.1.66
